### Descripción breve del proyecto
CredySmartJDL es una aplicación web estática que permite a los usuarios explorar, simular y solicitar créditos de diferentes tipos (vehículo, vivienda, educativo, libre inversión, entre otros).  
El sitio ofrece una interfaz limpia, responsive y amigable, diseñada para brindar una experiencia intuitiva al usuario final.  
Desarrollado con HTML, CSS y Bootstrap , el proyecto sigue buenas prácticas de diseño y accesibilidad.

###  Instrucciones para ejecutar el proyecto
1. Clona o descarga este repositorio.
2. Abre la carpeta del proyecto.
3. **Haz doble clic en el archivo index.html** para abrirlo en tu navegador web. 
4. Navega por las páginas:
   - *Inicio*: index.html
   - *Buscar*: Busqueda.html
   - *Solicitar crédito*: Solicitud.html

 Recomendado: usa *Google Chrome* o *Mozilla Firefox* para la mejor compatibilidad.
